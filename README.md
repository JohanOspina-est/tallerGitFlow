## Instalación
1. Ve al repositorio del profesor en github
2. Utiliza el acceso directo "Use this template", crearas tu propio repositorio
3. Clona el repositorio en tu maquina local: git clone https://github.com/TU_USUARIO/NOMBRE_REPO.git
4. Completa las misiones
## Uso
Practica sobre gitflow, este proyecto puedes utilizarlo para probar comandos basicos de git:
Por ejemplo:
- Ver estado(rama): git status
- Cambiar a otra rama: git checkout rombreRama
- Crear una rama nueva: git checkout -b nombreRama
- Subir rama al repositorio remoto: git push -u origin rama
